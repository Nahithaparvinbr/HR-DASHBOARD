# HR-DASHBOARD

I created this HR Dashboard using Power BI to analyze employee data and gain insights into workforce trends and organizational performance. The dashboard provides a clear overview of employee distribution, attrition, department performance, and key HR metrics. It helps in understanding employee behavior, monitoring retention rates, and supporting data-driven HR decisions.

Key Features

Total Employees overview
Employee Attrition analysis
Department-wise employee distribution
Gender-based employee insights
Job role and salary analysis
Experience and age group breakdown
Interactive filters for better analysis

Tools Used

Microsoft Excel – Data cleaninG, preparation, Pivot tables, charts, dashboard and calculations
Power BI – Dashboard design and data visualization
DAX – Calculated measures and KPIs

This project demonstrates my skills in data cleaning, data modeling, KPI development, and building interactive business dashboards using Power BI and Excel
